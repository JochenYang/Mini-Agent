# 小红书编纂智能体技能 (rednote-compilation)

专门为小红书内容创作者和营销人员设计的智能体技能，提供从内容策划到最终发布的一站式小红书编纂服务。

## 功能特性

### 🎯 核心功能
- **文案创作**：基于项目类型创作高质量小红书笔记正文
- **标题优化**：生成多种类型的爆款标题选项
- **标签推荐**：智能推荐相关标签提高曝光
- **图片配文**：为图片添加吸引人的配文
- **内容策划**：制定完整的内容策略和发布计划
- **爆款分析**：分析内容成功要素和优化建议

### 🛠️ 工具支持
- **内容分析器** (`scripts/analyzer.py`)：深度分析内容质量和优化方向
- **标题生成器** (`scripts/title_generator.py`)：基于产品信息和关键词生成多种标题
- **模板库**：丰富的内容模板和框架
- **标签库**：全面的标签推荐和组合策略
- **运营指南**：专业的小红书运营策略和方法

### 📚 资源库
- **内容模板**：美妆、美食、穿搭、生活、学习等各领域文案模板
- **运营策略**：平台算法解析、用户画像分析、竞品分析等
- **项目模板**：产品推广、服务宣传、知识分享、生活方式等各类项目模板

## 使用方法

### 快速开始
1. **明确项目类型**：告诉技能你的具体项目是什么
2. **描述目标受众**：说明想要吸引的用户群体
3. **确定内容目标**：种草、测评、分享、推广等
4. **提供基础信息**：产品特色、优势、卖点等

### 详细使用
#### 文案创作流程
1. 分析项目类型和特点
2. 研究目标受众需求和痛点
3. 确定内容角度和表达方式
4. 运用爆款文案框架创作
5. 优化语言表达和情感渲染

#### 标题优化流程
1. 分析原标题的问题
2. 研究同类型爆款标题
3. 选择合适的标题策略
4. 生成多个标题选项
5. 提供选择建议和使用场景

#### 标签推荐流程
1. 分析内容关键词和主题
2. 选择垂直领域标签
3. 组合通用热门标签
4. 添加长尾精准标签
5. 控制标签数量和密度
