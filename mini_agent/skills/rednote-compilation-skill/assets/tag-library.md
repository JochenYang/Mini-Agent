1. **数量控制**：每条内容使用8-12个标签最佳
2. **相关性**：确保标签与内容高度相关
3. **通用+垂直**：结合通用热门标签和垂直领域标签
4. **时效性**：关注当前热点话题标签
5. **避免堆砌**：不要重复使用相似标签

### 标签组合策略
1. **头部标签**：2-3个通用大标签（如#小红书 #种草）
2. **中部标签**：4-5个垂直领域标签（如#美妆博主 #护肤品推荐）
