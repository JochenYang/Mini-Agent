# 小红书编纂智能体技能

## 技能概述

这个技能专门为小红书内容创作者和营销人员设计，提供从内容策划到最终发布的一站式编纂服务。基于小红书平台特性和用户行为模式，帮助创作爆款内容。

## 使用时机

当用户需要以下服务时使用此技能：
- 为产品或服务创作小红书推广文案
- 优化现有内容的标题和标签
- 为图片添加吸引人的配文
- 进行小红书内容策划和爆款分析
- 基于具体项目类型制定内容策略

## 核心功能

### 1. 文案创作（笔记正文）

根据项目类型和目标受众创作小红书笔记正文：

#### 产品推广类
- **美妆护肤**：产品测评、使用心得、对比分析
- **数码产品**：功能介绍、使用体验、购买建议
- **生活用品**：实用评测、生活场景应用

#### 服务宣传类  
- **教育培训**：课程介绍、学习心得、技能提升
- **健身服务**：训练效果、教练评价、课程体验
- **咨询服务**：服务流程、案例分享、客户见证

#### 知识分享类
- **技能教程**：步骤详解、注意事项、进阶技巧
- **经验总结**：心路历程、避坑指南、实用建议
- **攻略指南**：全面攻略、实用工具、节省技巧

#### 生活方式类
- **美食制作**：制作过程、口感描述、搭配建议
- **旅行攻略**：路线规划、景点介绍、实用Tips
- **家居装修**：装修过程、设计理念、预算控制

### 2. 标题优化

提供多种标题策略：
- **情感驱动型**：引发共鸣、戳中痛点
- **数字标题型**：具体数据、量化效果
- **好奇引导型**：悬念设置、制造悬念
- **对比冲突型**：前后对比、冲突对比
- **权威背书型**：专家推荐、专业认证
- **限时紧迫型**：限时优惠、稀缺性

### 3. 标签推荐

根据内容类型推荐相关标签：
- **通用热门标签**：提高曝光率
- **精准领域标签**：吸引目标受众
- **长尾标签**：提高搜索精准度
- **品牌标签**：配合品牌推广
- **地点标签**：本地化推广

### 4. 图片配文

为单张或多张图片添加吸引人的配文：
- **单图配文**：主图重点突出，细节补充
- **多图配文**：图片故事线，逻辑清晰
- **图文呼应**：图片与文字完美配合
- **视觉引导**：突出重点，引导关注

### 5. 内容策划

制定完整的内容策略：
- **内容矩阵**：主题规划、发布节奏
- **用户画像**：目标受众分析
- **竞品分析**：同类内容研究
- **效果预期**：预期效果和KPI设定

### 6. 爆款分析

分析内容成功要素：
- **热门元素识别**：成功内容的共同特征
- **用户行为分析**：用户互动模式
- **时机把握**：最佳发布时间和话题
- **迭代优化**：基于数据的持续改进

## 使用方法

### 快速开始

1. **明确项目类型**：告诉技能你的具体项目是什么
2. **描述目标受众**：说明想要吸引的用户群体
3. **确定内容目标**：种草、测评、分享、推广等
4. **提供基础信息**：产品特色、优势、卖点等

### 详细流程

#### 文案创作流程
1. 分析项目类型和特点
2. 研究目标受众需求和痛点
3. 确定内容角度和表达方式
4. 运用爆款文案框架创作
5. 优化语言表达和情感渲染

#### 标题优化流程
1. 分析原标题的问题
2. 研究同类型爆款标题
3. 选择合适的标题策略
4. 生成多个标题选项
5. 提供选择建议和使用场景

## 内容框架模板

### 种草类文案框架
```
开头：痛点引入 + 解决方案预告
中间：产品详细介绍 + 使用体验 + 效果展示
结尾：总结推荐 + 互动引导
```

### 测评类文案框架  
```
开头：产品背景 + 测评目的
中间：详细测评过程 + 多角度分析 + 优缺点总结
结尾：购买建议 + 使用场景推荐
```

### 分享类文案框架
```
开头：话题引入 + 个人经验背景
中间：具体分享内容 + 实用tips + 注意事项
结尾：经验总结 + 互动讨论引导
```

### 教程类文案框架
```
开头：教程目的 + 适用人群 + 预期效果
中间：详细步骤 + 技巧说明 + 常见问题
结尾：进阶建议 + 成果展示 + 学习资源推荐
```
